## Testing Content for {{PARTNER_NAME}}'s Voice

Ask these questions:

1. **Would {{PARTNER_NAME}} say this?** Read it out loud. Does it sound like them?
2. **Is it clear?** Can the audience read it once and understand?
3. **Is it direct?** Does it get to the point without hedging?
4. **Is it helpful?** Does it give actionable insight?
5. **Is it appropriate?** Does it respect the audience's expertise?

If "no" to any of these, revise before submitting.
