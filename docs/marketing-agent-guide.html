The Intelligence Loop

Each session reads enriched context, executes your marketing tasks, then writes back intelligence to docs/intelligence/internal/. Four files accumulate automatically:

  • context-intelligence-ledger.md — session history and decisions across every conversation
  • competitive-intelligence-tracking.md — competitor moves and market positioning updates
  • performance-analysis-history.md — content and campaign performance metrics over time
  • seasonal-patterns.md — seasonal trends and timing insights for future planning

The next session inherits this accumulated knowledge, creating a continuously improving marketing engine.

Cadence Tips

  • ● Intelligence first. Start the week with /intel to feed fresh data into all downstream tools.
  • ● Process calls promptly. Run /transcript after any client or partner call to capture insights while fresh.
  • ● Batch images. Queue up image generation on quieter days rather than one-off during content creation.
  • ● Monthly deep dives. Run /analyst with comprehensive analysis once a month for strategic recalibration.
  • ● Periodic health checks. Run /health monthly to catch stale intelligence data or missing configuration before it affects output quality.
