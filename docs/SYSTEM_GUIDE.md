## ‚ö†Ô∏è Troubleshooting & Best Practices

### Common Issues & Solutions

**Issue**: CMO agent shows "Business Context Configuration Required"
**Solution**: Run `/onboard` command to configure {{PLACEHOLDER}} values in CLAUDE.md
**Prevention**: Always complete onboarding before strategic planning

**Issue**: Dashboard not loading (Port 3000)
**Solution**: Run `./automation/server_manager.sh start` to launch servers
**Prevention**: Use `./automation/server_manager.sh status` to monitor server health

**Issue**: Content validation fails
**Solution**: Check brand compliance and run `./automation/content-validation-hook.sh`
**Prevention**: Use quality assurance pathway (`*quality`) for high-stakes content

**Issue**: Agent coordination errors
**Solution**: Verify all agent files exist in `.claude/agents/` directory
**Prevention**: Regular system testing using `./automation/test_agents.sh`

**Issue**: Airtable integration failing
**Solution**: Verify API credentials and run `./automation/test_integration.sh`
**Prevention**: Monitor integration health in dashboard analytics view

### Best Practices

**Strategic Planning**:
- Always start with `/onboard` for new implementations
- Use `/cmo` for strategic pathways, `/analyst` for performance optimization
- Present numbered options to users for clear decision-making

**Content Management**:
- Leverage LOCAL generation for immediate text content needs
- Use SYSTEMATIC generation for multi-platform, formatted content
- Implement quality validation for all high-visibility content

**System Maintenance**:
- Monitor server health regularly using `./automation/server_manager.sh status`
- Update brand configuration as business evolves
- Test agent coordination periodically with `./automation/test_agents.sh`

**Performance Optimization**:
- Use AEO pathway for AI search engine optimization
- Implement A/B testing for campaign optimization
- Monitor competitive landscape quarterly
