# Section Patterns

Templates and structures for common landing page sections.

## Hero Section

The first thing visitors see. Must communicate value instantly.

### Hero Components

| Element | Purpose | Priority |
|---------|---------|----------|
| Headline | Core value proposition | Required |
| Subheadline | Supporting detail/credibility | Required |
| Primary CTA | Main conversion action | Required |
| Hero Image/Video | Visual reinforcement | Recommended |
| Trust badges | Quick credibility | Optional |

### Hero Layouts

**Centered Hero:**
```
[Logo/Nav]

        [Headline - Centered]
      [Subheadline - Centered]
         [CTA Button]
    [Trust logos or badges]

        [Hero Image/Video]
```
Best for: Simple, focused message

**Split Hero (Text Left, Image Right):**
```
[Logo/Nav]

[Headline]            [Hero Image]
[Subheadline]
[CTA Button]
[Trust badges]
```
Best for: Product visuals, SaaS, apps

**Video Background Hero:**
```
[Overlay gradient]
        [Headline - Large, Centered]
          [Subheadline]
           [CTA Button]
[Background video - muted, looping]
```
Best for: Lifestyle brands, events, emotional appeal

### Hero Copy Pattern

```
Headline: [Primary benefit or transformation]
Subheadline: [How you deliver it] + [for whom] + [trust signal]
CTA: [Action] + [Benefit/Qualifier]
```

Example:
```
Headline: Launch Your Course in a Weekend
Subheadline: Our drag-and-drop builder lets creators go from idea to income
             without any tech skills. Trusted by 50,000+ instructors.
CTA: Start Building Free
```

## Problem Section

Agitate the pain to create motivation for the solution.

### Problem Section Structure

```
[Section Headline: Acknowledge the problem]

[Problem Statement 1]    [Problem Statement 2]    [Problem Statement 3]
[Icon]                   [Icon]                   [Icon]
[Pain point]             [Pain point]             [Pain point]
[Consequence]            [Consequence]            [Consequence]

[Optional: Transition to solution]
```

### Problem Copy Patterns

**Direct acknowledgment:**
> "You're tired of [pain point]. You've tried [alternatives]. Nothing works."

**Cost of inaction:**
> "Every day you spend [struggling with problem] is another day you lose [opportunity]."

**Empathy + agitation:**
> "We get it. [Problem] is frustrating. And the worst part? [Deeper consequence]."

## Solution/Benefits Section

Show the transformation your product enables.

### Benefits Structure

**3-Column Layout:**
```
[Section Headline: The outcome they want]

[Icon/Image 1]          [Icon/Image 2]          [Icon/Image 3]
[Benefit Title]         [Benefit Title]         [Benefit Title]
[2-3 sentences          [2-3 sentences          [2-3 sentences
describing outcome]     describing outcome]     describing outcome]
```

**Alternating Layout:**
```
[Image Left]    [Benefit Title]
                [Benefit Description]
                [Supporting detail]

                [Benefit Title]    [Image Right]
                [Benefit Description]
                [Supporting detail]
```

### Benefit Copy Pattern

```
[Benefit Title: Outcome they achieve]
[Description: How it works + why it matters]
[Proof point or specific detail]
```

Example:
```
Automate Your Outreach
Send personalized sequences to hundreds of prospects while you focus on
closing deals. Our AI personalizes each message based on prospect data.
Average time saved: 10 hours per week.
```

## How It Works Section

Reduce complexity, show the path to success.

### 3-Step Process

```
[Section Headline: How It Works / Getting Started is Easy]

Step 1              Step 2              Step 3
[Number/Icon]       [Number/Icon]       [Number/Icon]
[Title]             [Title]             [Title]
[Brief description] [Brief description] [Brief description]

[CTA: Get Started]
```

### Timeline/Journey

```
[Section Headline: Your Path to [Outcome]]

○ [Step 1] ————— ○ [Step 2] ————— ○ [Step 3] ————— ★ [Result]
  [Details]         [Details]         [Details]         [Outcome]
```

### Process Copy Pattern

```
Step 1: [Action verb + simple task]
[1-2 sentences explaining what they do and how easy it is]

Step 2: [Action verb + next task]
[1-2 sentences explaining progression]

Step 3: [Action verb + outcome task]
[1-2 sentences emphasizing the result they achieve]
```

## Social Proof Section

Build trust through evidence of others' success.

### Logo Bar

```
[Optional: "Trusted by leading teams"]

[Logo] [Logo] [Logo] [Logo] [Logo] [Logo]
```

Best placement: Just after hero or early in page

### Testimonial Grid

```
[Section Headline: What Our Customers Say]

[Testimonial Card 1]    [Testimonial Card 2]    [Testimonial Card 3]
"Quote..."              "Quote..."              "Quote..."
[Photo] [Name]          [Photo] [Name]          [Photo] [Name]
[Title, Company]        [Title, Company]        [Title, Company]
```

### Featured Testimonial

```
[Large quote mark or design element]

"[Longer, more detailed testimonial quote about transformation
and results achieved.]"

[Photo]
[Name]
[Title, Company]
[Optional: Star rating or metric]
```

### Metrics/Stats Bar

```
[Stat 1]              [Stat 2]              [Stat 3]
[Large Number]        [Large Number]        [Large Number]
[Description]         [Description]         [Description]
```

Example:
```
10,000+               4.9/5                 50M+
Happy customers       Average rating        Messages sent
```

## Final CTA Section

Last push before footer.

### Simple Final CTA

```
[Background: Different color or image]

[Compelling headline restating value]
[Brief supporting text or urgency element]

[Large CTA Button]

[Trust element: guarantee, security, or testimonial snippet]
```

### CTA with Form

```
[Headline: Ready to [achieve outcome]?]

[Input: Email]
[CTA Button: Get Started]

[Privacy note and/or trust badges]
```

### Split CTA (Two Options)

```
[Headline]

[Option 1: Self-serve]        [Option 2: Talk to sales]
[CTA: Start Free Trial]       [CTA: Book a Demo]
[Supporting text]             [Supporting text]
```

## Guarantee Section

Reduce risk to encourage action.

### Guarantee Block

```
[Guarantee Badge/Icon]

[Bold: 30-Day Money-Back Guarantee]

[Explanation of guarantee terms in friendly language.
Emphasize that they have nothing to lose.]

"Try it risk-free. If you're not completely satisfied,
we'll refund every penny. No questions asked."
```

## Section Order Recommendations

### Lead Generation Page
1. Hero (headline, subhead, CTA, form)
2. Problem
3. Solution/Benefits
4. Social Proof (logos + testimonials)
5. How It Works
6. FAQ
7. Final CTA

### SaaS Product Page
1. Hero
2. Logo bar
3. Features with screenshots
4. Benefits (outcomes)
5. How It Works
6. Social Proof
7. Pricing
8. FAQ
9. Final CTA

### E-commerce Product Page
1. Hero/Product display
2. Benefits
3. Social Proof (reviews)
4. Features/Specs
5. Comparison
6. Guarantee
7. Urgency/Scarcity
8. CTA
